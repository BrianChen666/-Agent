# 售后政策

## 适用范围

本政策适用于虚构型号 X100 滚筒洗衣机、X200 洗烘一体机、W80 波轮洗衣机。实际项目中可将本文件作为售后政策类知识源，供 Agent 判断保修、收费、安装、退换货和发票要求。

## 保修范围

1. 整机保修 1 年，自有效购买凭证日期起算。
2. 主要部件保修 3 年。主要部件包括电机、主控板、显示板、进水阀、排水泵或排水阀、水位传感器、门锁或上盖锁、X200 烘干风机与烘干加热组件。
3. 保修仅覆盖正常家庭使用条件下的非人为质量故障。
4. 保修期内经售后检测确认为产品质量故障，免收基础上门检测费、人工费和对应保内配件费。
5. 未提供购买凭证时，以机身铭牌或系统记录的出厂日期推算保修期。
6. 同一故障维修后，更换的保内配件随整机剩余保修期继续保修；若剩余期不足 90 天，按 90 天计算。

## 非保修范围

以下情况不属于免费保修：

1. 超过整机或主要部件保修期限。
2. 人为损坏，包括摔碰、进水、私自拆修、异物进入、强行开门、短接门锁、改装线路。
3. 非家庭使用场景，例如宿舍集中洗衣房、酒店、洗衣店、工厂、出租房高频商用。
4. 未按说明书要求安装、使用或维护，例如排水管高度严重不当、长期不清理过滤器导致堵塞损坏。
5. 使用环境不符合要求，例如电压不稳、插座无接地、水压长期异常、潮湿腐蚀环境。
6. 消耗品和外观件自然磨损，例如进水管、排水管、过滤网、洗涤剂盒、门封圈外观污染、面板划痕。
7. 不可抗力造成的损坏，例如火灾、水灾、雷击、地震。
8. 用户无法提供机器序列号，且铭牌被撕毁、涂改或无法识别。

## 上门检测费

1. 保内质量故障：免收基础上门检测费。
2. 保内但检测为非产品质量原因：可收取检测费或上门费。
3. 保外维修：收取上门检测费、人工费和配件费。检测后当场维修的，检测费可按当地服务规则抵扣部分人工费。
4. 偏远地区、海岛、山区、跨区服务可能产生额外上门交通费用，预约前需告知用户。
5. 夜间、节假日加急服务可能产生加急费用，以用户确认后的预约单为准。

## 安装服务

1. 新机首次基础安装免费，需提供有效购买凭证或系统订单。
2. 基础安装包括拆箱、摆放、调平、连接原有合规水龙头、连接排水管、通电试机和基础使用说明。
3. 需要打孔、改造水电、加装专用水龙头、延长进排水管、购买支架或额外辅材时，材料和特殊施工费用由用户承担。
4. 用户自行安装导致漏水、进水异常、排水异常、机身倾斜或运输螺栓未拆引发震动的，检测后可判定为非产品质量问题。
5. 移机、二次安装、跨城市安装不属于免费首次安装，按当地服务报价执行。

## 退换货条件

1. 自签收或安装完成之日起 7 日内，经售后检测确认存在产品质量故障，用户可申请退货、换货或维修。
2. 自签收或安装完成之日起 15 日内，经售后检测确认存在产品质量故障，用户可申请换货或维修。
3. 超过 15 日且在保修期内，原则上按维修处理。
4. 退换货需保持主要附件齐全，包括进水管、排水管、运输螺栓、说明书、保修卡和发票或订单凭证。
5. 因安装环境、排水地漏、外部水压、电源条件、使用不当、外观磕碰或人为拆修造成的问题，不适用质量退换货。
6. 大件商品退换货需售后上门检测并出具检测结论，客服不得仅凭用户描述承诺退换。

## 发票和购买凭证要求

1. 有效购买凭证包括纸质发票、电子发票、官方商城订单、授权渠道订单截图。
2. 凭证应包含购买日期、商品名称或型号、销售方信息。
3. 发票抬头不影响产品保修，但需能证明购买时间和商品。
4. 无凭证时以出厂日期推算保修期，可能导致保修期缩短。
5. 凭证与机身序列号不一致时，需人工核验。

## 配件收费规则

1. 保内质量故障更换主要部件不收取配件费。
2. 保外维修按配件目录收费，并叠加人工费和可能的上门费。
3. 因人为原因导致的门封圈破损、排水管破裂、过滤器丢失、面板进水等，按非保修配件收费。
4. 更换下来的保外旧件原则上可交还用户；涉及返厂检测的配件需按服务单说明处理。
5. 维修前需向用户说明预计费用范围，用户确认后再派工或维修。

## Agent 判断规则

1. 不要承诺“必然免费”或“必然退换”，应使用“需售后检测确认”。
2. 保修判断必须同时确认购买日期、凭证、故障性质、使用场景和是否人为拆修。
3. 涉及费用争议、退换货争议、凭证缺失、偏远地区上门费时，建议转人工。
4. 安全风险高于政策解释。漏水、跳闸、焦味、冒烟、门锁异常时，先给安全处置，再解释保修。
